## Transform **GRanges** to **Data.Frame**

```
cnvdf <- data.frame(chr=as.character(seqnames(mycnv)), start=start(mycnv), end=end(mycnv))
callcnv <- mcols(mycnv)
cnvdf <- cbind(cnvdf, callcnv)
```
